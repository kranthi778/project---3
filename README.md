# Footprinting and Reconnaissance Project

I am working on a hands-on Footprinting and Reconnaissance project based on **CEH v13 – Module 02**. This Footprinting and Reconnaissance project is about using Open Source Intelligence techniques to gather information about a target before we do any security testing.

The Footprinting and Reconnaissance project covers a lot of things like domain intelligence, DNS footprinting, website reconnaissance Google Dorking, email and social media footprinting and metadata analysis using industry- tools and techniques. We use these tools and techniques to gather available information about a target.
